# Pomodoro Pride

A sleek Pomodoro timer Firefox extension themed with pride flag gradients. Pick your flag, stay focused, and track your progress — all with a frosted-glass UI that looks beautiful on every flag.

## Features

**Timer**
- 25-minute focus sessions, 5-minute short breaks, 15-minute long breaks
- Timer persists in the background — close the popup and it keeps running
- Toolbar badge shows remaining minutes at a glance
- Ding notification when a session completes

**35 Pride Flags**
- Classic Rainbow, Progress, Trans, Bisexual, Pansexual, Lesbian, Gay (MLM), Sapphic, Achillean, Nonbinary, Asexual, Aromantic, Aroace, Demisexual, Demiromantic, Genderfluid, Genderqueer, Intersex, Agender, Polysexual, Omnisexual, Polyamorous, Demiboy, Demigirl, Bigender, Bear, Abrosexual, Pangender, Trigender, Neutrois, Maverique, Aceflux, Aroflux, Androsexual, Gynesexual

**Customization**
- Choose any pride flag as your theme — the entire UI adapts
- Custom text color picker if you want to override the auto-detected color
- Flags render as accurate hard-edged stripes with correct proportions

**Stats**
- All-time totals: lifetime pomodoros, focus hours, current streak, best streak
- Weekly activity heat map with navigation to any past week
- Calendar date picker to jump to any week
- Daily breakdown: pomodoros, focus time, short breaks, long breaks
- Session log with timestamps
- Export/import stats as JSON
- Reset stats with confirmation

## Install

### From Firefox Add-ons (recommended)
*Coming soon*

### From Source
1. Clone this repo
2. Open Firefox and navigate to `about:debugging#/runtime/this-firefox`
3. Click **Load Temporary Add-on**
4. Select the `manifest.json` file from the project folder
5. The extension icon appears in your toolbar — click it to start
